How to use your App Store promo code

If you received a promo code for one of our iPhone or iPad apps, redeem it in the App Store and the app will be added to the Apple Account you are currently signed in with.

Redeem your code

  1. Open the App Store on your iPhone or iPad.
  2. Tap your profile photo or the sign-in button at the top of the screen.
  3. Tap Redeem Gift Card or Code.
  4. If prompted, sign in with the Apple Account you want to use for the app.
  5. Tap Enter Code Manually and type in the promo code you received.
  6. Tap Redeem.
  7. Once the code is accepted, download the app from the App Store if it does not begin installing automatically.

Before you start

Use the right Apple Account The redemption will be tied to the Apple Account that is signed in when you redeem.
Manual entry is usually best Promo codes are easiest to redeem by typing them directly instead of scanning.
Region matters Apple codes can fail if the code and App Store account are set to different countries or regions.

Troubleshooting

  • If you do not see Redeem Gift Card or Code, sign in first and then check again.
  • Make sure the code is entered exactly as provided.
  • If the code says it has already been redeemed, confirm you are signed in with the intended Apple Account and check your previous purchases.
  • If the code is tied to another storefront, switch to the correct App Store region or contact the sender.

These steps are based on current Apple Support guidance for redeeming codes in the App Store on iPhone and iPad.
