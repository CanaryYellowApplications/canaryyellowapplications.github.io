The Set Lighting App Set Lighting pocket resource. All the digital tools the modern Set Lighting Technician will need in the field and back in the truck. Databases! Formulas! Diagnostics! Reference Materials!
ScreenGlow Mobile Mobile device luminaire. Fully functional tool for filmmakers to use their iOS device screen as a lighting device. Useful for both illumination from the display and as a screen chroma key. Modes include CCT, RGB, HSL, Party FX, TV FX, and DMX control via Art-Net or sACN traffic on the connected network.
ScreenGlow Desktop Desktop Screen Luminaire. Fully functional tool for filmmakers to use their Mac screen as a lighting device. Useful for both illumination from the display and as a screen chroma key. Modes include CCT, RGB, HSL, Party FX, TV FX, and DMX control via Art-Net or sACN traffic on the connected network. Overlays include tracking marks and corner marks, plus display masks at various aspect ratios.
LuxLog Lighting Setup Logger. Keep track of every detail of your lighting setup. Includes inputs for illuminant color, brightness, DMX, modifiers, and positions, as well as camera settings, attachments, and more.
FalseColor+ Image analytics. Analyze and export any image using tools like False Color, Zebra, and Histogram.
